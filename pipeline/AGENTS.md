# Agent-Guide für carclip

Dieses Repo ist 'n clean Demo – nur Platzhalter und Dummies.  
Kein echter Code, keine Keys, keine Daten.  

**Was du darfst:**  
- Platzhalter-Skripte ersetzen (z. B. autoverkauf.sh, run.sh)  
- config.sh anpassen (VOICES, FONTS usw.)  
- Pull Requests für Verbesserungen (z. B. besserer TTS-Fallback)  

**Was du NICHT darfst:**  
- Echte API-Keys, Pfade oder Logs einchecken  
- Persönliche Daten (Kennzeichen, Preise, IDs) pushen  
- .gitignore ignorieren – Logs, .cache, .tmp bleiben draußen  

**Ziel:** Jeder kann's lokal testen, ohne Risiko.  
Wenn du 'ne echte Pipeline baust: Fork & go – aber lass das Original clean!
